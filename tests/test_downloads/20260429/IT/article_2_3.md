# 情報やシステム守る 基本の3つの視点 | 日経クロステック（xTECH）

新年度からセキュリティーを担当するように言われた。情報システムを防御する役割というのは分かる。でも、そもそもセキュリティーとは一体何から手をつけたらいいのだろう――。

　セキュリティーというと漠然と情報システムを悪意ある攻撃者から守ることだと考えるだろう。それ自体は誤りではない。実際、近年は企業が事業を運営する上で情報システムは必要不可欠な存在である。

　このため企業システムは事業継続を揺るがしたり、金銭的な損害が発生したりするようなサイバーリスクにさらされている。2025年はアサヒグループホールディングスやアスクルといった大手企業がランサムウエア攻撃を受けて事業が一時的に停止する事態となった。

　他にも社員が端末を紛失して情報を流出させたり、内部不正を働いて機密情報を盗み出したりするケースもある。

　こうしたインシデントではデータの流出や人質化などに焦点が当たりがちだが、守るべき対象はデータだけではない。「データに偏った守りでは不十分。社員や機器、情報、企業ブランドなどのリソースを守る必要がある」（NRIセキュアテクノロジーズの時田剛セキュリティ人材育成室長、取材当時）。
