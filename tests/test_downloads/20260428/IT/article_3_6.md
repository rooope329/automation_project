# Googleの「Pixel 10a」、1世代前と同じ価格で同じチップ搭載は買いか？ | 日経クロステック（xTECH）

米Google（グーグル）が2026年も、スマートフォン「Google Pixel」シリーズの最廉価モデル「Google Pixel 10a」を投入した。毎年夏の終わりごろに「a」が付かない新モデルが登場し、翌春にその廉価版に当たる「Pixel a」シリーズが発売されるのが流れだ。Google Pixel 10も2025年8月にリリースされている。

　Google Pixel 10aの価格は7万9900円（税込み、以下同）。ライバルである米Apple（アップル）のiPhoneの最廉価モデル「iPhone 17e」が9万9800円であることを考えると、2万円ほど安いところがのが魅力だ。

　ということで、今回は日本限定モデルであるIsai Blueをレビューしていく。

#### 軽くてシンプルなのが良い

　Google Pixel 10aは、6.3インチの一般的なサイズのモデルで、比較的持ちやすい。ただし、幅は73ミリメートルなので、Google Pixel 10の72ミリより1ミリ太い。画面サイズの割にはスリムなわけではない。

　これは、額縁が太いためだろう。額縁の太さは個人的にはかなり気になるが、見た目の問題だ。なお、画面は四隅がかなり丸いので、アプリによっては各種機能ボタンなどが使いづらいかもしれない。

　重量はカタログ値で183グラムと軽く、Google Pixel 9aより3グラム軽い。また、上位機のGoogle Pixel 10は204グラムなので大幅に軽量だ。これは、カメラの作りが大きく違う点もありそうだ。Google Pixel 10aは、カメラの出っ張りがなく、スマートなデザインだ。

#### 性能は据え置きなのがポイント

　Google Pixel 10aは、Google Pixel 9aと同じ価格である。つまり、据え置きなわけだ。これで、チップが進化しているのなら素晴らしいのだが、実はチップも据え置きで、「Google Tensor G4」だ。

　ちなみに、Google Pixel 10は「Google Tensor G5」となる。つまり、Google Pixel 9aまでは、前年の夏の終わりごろのモデルと同じチップを採用していた。ところが、Google Pixel 10aは1世代前のチップになるのだ。

　これは、価格を抑えるための方策なのか、機種の差を明確にするためなのかは不明だ。しかし、Google Pixel 9aから買い替える理由はほとんど見当たらない。

　実際に使ってみても性能は十分で、中級クラスのスマホとしては満足できる。ただし、ヘビーなゲームを3～4年先まで使い続けるのは厳しそうだ。PixelシリーズはGPU性能よりもAI（人工知能）性能に注力しているようで、ゲームにおすすめの機種とは言えなくなってきている。

#### カメラは普通にきれいだが望遠に弱い

　カメラは48メガピクセルの標準カメラと、13メガピクセルの超広角となる。どちらも、非常に美しく、普通に撮影して文句なし。僕が所有しているのは、Google Pixel 10 Pro XLだが、標準と超広角の写真はほとんど差が分からない。

　ネックなのは、望遠機能が非搭載な点だ。ズームは最大8倍だが、すべてデジタルズームだ。個人的な感覚では2～3倍までは実用的だが、そこを超えると荒れて見える。最大の8倍はぼんやりして使いたくない。

　ちなみに、Google Pixel 10 Pro XLは最大100倍ズームに対応しており、30倍以上でも画質がさほど荒れる印象はない。そう考えると、標準と超広角は満足でき、望遠も2～3倍までの利用なら大満足だと思う。

#### AIはかなり楽しく便利に使える

　Google Pixel 10aのAI機能は盛りだくさんで、いろいろ便利に利用できる。以前は消しゴムマジックが話題だったが、最近は被写体の一部を移動できる。AIによる機能だが、なかなか実用的だ。

　音声消しゴムマジックで動画の騒音を消せるし、おなじみの「かこって検索」やGemini Liveも利用可能。レコーダーアプリによるテキスト化や音楽の作成、リアルタイム翻訳も重宝する。

　ただし、上位機で使える「Pixelスクリーンショット」や「マジックサジェスト」には非対応だ。Pixelシリーズは、AIに強いのが魅力で、韓国Samsung Electronics（サムスン電子）のGalaxy S26シリーズと激しく競っている。どちらを手に入れてもAIでは満足できそうだ。

　Google Pixel 10aは、中級クラスのスマホとしてはおすすめの1台で欠点は少ない。カメラが納得できるなら買いだろう。
