# 「次の段階に踏み出す」、Google CloudがAIエージェントの統合基盤を投入 | 日経クロステック（xTECH）

米Google（グーグル）のクラウドサービス部門であるGoogle Cloudは2026年4月22日（米国時間）、AI（人工知能）エージェントの構築や管理などを包括的に支援する「Gemini Enterprise Agent Platform」を発表した。企業がAIエージェントを導入・活用するために必要な機能をまとめた統合基盤と位置付ける。

　Google Cloudが米ラスベガスで開催中の年次イベント「Google Cloud Next '26」で明らかにした。同社のトーマス・クリアンCEO（最高経営責任者）は4月22日の基調講演で「グーグルのAIを全ての従業員とワークフローに導入するために、次の段階に踏み出す」と語った。

　クリアンCEOらが登壇した4月22日 の基調講演は、AIエージェント一色だった。目玉は、開発者向けに展開するGemini Enterprise Agent Platformだ。AI開発基盤である「Vertex AI」の進化形とし、AIエージェントの構築や拡張、ガバナンス、最適化に関する機能をまとめて提供する。

　例えば、ローコードでAIエージェントを開発できる「Agent Studio」が構築フェーズを支援する。ガバナンスという観点では、「Agent Identity」「Agent Registry」「Agent Gateway」を一体的に運用することで、全てのAIエージェントが追跡可能なIDを保有し、セキュリティーが担保された環境内で動作することを支援するという。

　幅広いユーザーがAIエージェントを活用するための入り口になるのが「Gemini Enterprise app」だ。具体的には、いくつかのシステムにまたがるプロセスを自動化するエージェントを開発できる「Agent Designer」や、アクティブなAIエージェントを監視し、設定を見直したり、メールやチャット経由で通知をリアルタイムに受け取ったりできる「Inbox in Gemini Enterprise」といった機能を備える。
