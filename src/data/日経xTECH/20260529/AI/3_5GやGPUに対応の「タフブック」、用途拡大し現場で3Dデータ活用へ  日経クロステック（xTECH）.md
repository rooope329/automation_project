# 5GやGPUに対応の「タフブック」、用途拡大し現場で3Dデータ活用へ | 日経クロステック（xTECH）

　パナソニック コネクト（東京・中央）が、過酷な現場で使える高耐久ノートパソコン（PC）の用途を拡大している。5G（第5世代移動通信システム）に対応した他、GPU（画像処理半導体）を搭載して処理能力を向上。屋外環境での円滑なオンライン会議や、3Dデータを使った設計検討を実現して、新規需要の獲得を狙う。

　2026年6月から高耐久ノートPCブランド「TOUGHBOOK（タフブック）」の新機種として、「FZ-56」シリーズを投入する。通信機能や演算性能を高めつつ、従来と同等の耐久規格IP53に準拠した。粉じんや水しぶきにさらされる建築現場のような過酷な環境でも使用できる（**図1**）。

おからの粉末をFZ-56に吹きかけての防じん性能をアピールした。（写真：日経ものづくり）

　「作業現場からオンライン会議に参加したい」といった需要にも応える。FZ-56シリーズでは5Gに対応したモデルを用意し、通信容量の大きい用途での使い勝手を高めた。足元では、人工衛星による通信インフラの拡張などで、海洋上や山間部でも通信できる場面が増えている。屋外でも快適にオンライン会議ができるよう、スピーカーの配置を両側面から手前に変更して音の指向性を使用者側に向けたほか、マイクにはノイズキャンセリング機能を備えるなど、設計にも変更を加えた。
