# 「MINORIのこの状況を、いつまでも続けるのか……」 | 日経クロステック（xTECH）

![](https://cdn-xtech.nikkei.com/atcl/nxt/info/18/00061/052700091/_h260-1.jpg?__scale=w:680,h:233&_sh=03c0530920)

　みずほ銀行はゴールデンウイーク中の5月2日から3日にかけて、勘定系システム「MINORI」の更改作業を実施しました。MINORIは2019年に全面稼働しており、保守期限が到来するハードウエアを入れ替えるものです。更改対象のシステムが広範にわたるため、更改は2025年10月から2026年10月まで約1年をかけ、6回に分けて実施します。今回は第2弾で、ATM（現金自動預け払い機）や営業店端末をそれぞれ束ねるサーバーなどを入れ替えました。

　更改に当たり同行からは、作業の進め方や万一の場合に備えたフォールバックの段取りなども含めて事前に報道各社への情報提供があり、さらに更改作業中もメールで進捗状況の定期連絡を受け、「ここまで手厚く情報提供してくれるのか」と驚かされました。2021～22年に相次いだシステム障害の教訓という面も当然あるとは思いますが、最前線で更改の実務を担う同行IT部門の実直さがうかがえるものだと、私は感じました。

　さて、そんなMINORIについて、みずほフィナンシャルグループ（FG）の木原正裕社長が5月19日の記者会見で興味深いコメントをしています。「今は更改作業をきちんとやることに徹底的に集中している」と前置きを入れつつ、先の展望として「このMINORIというもの、この状況をいつまでも続けるかどうか、将来に向けてもっと性能の高いものがないかは、絶えず考えるべき話だ。メインフレームか、オンプレミスか、クラウドなのかは置いておいても、未来に向けて、次の我々のシステムがどういうものになっていくのがよいかを考えなくてはいけない」と述べました。
