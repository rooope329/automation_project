# AWSガーマンCEO「可能性解き放つ」　オープンAIの最新AIモデル提供で | 日経クロステック（xTECH）

米アマゾン・ウェブ・サービス（AWS）のマット・ガーマンCEO（最高経営責任者）はこのほど、AWSのイベントで日経クロステックなどの取材に応じ、生成AI（人工知能）サービス「Amazon Bedrock」を通じて、米オープンAIの最新AIモデルを提供することについて「大きな可能性を解き放つ」と語った。
