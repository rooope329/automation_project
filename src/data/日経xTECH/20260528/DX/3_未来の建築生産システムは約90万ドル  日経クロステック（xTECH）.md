# 未来の建築生産システムは約90万ドル | 日経クロステック（xTECH）

独自開発の建設3Dプリンターで、これまでに約250棟の住宅などを建ててきたスタートアップの米ICON（アイコン）が、いよいよプリンターの販売に乗り出す。未来の建築生産システムのお値段は89万9000ドル（約1億4200万円）だ。
